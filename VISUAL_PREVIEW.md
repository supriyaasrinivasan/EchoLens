## Interactive States

### Normal State
```
┌─────────────┐
│   📚        │
│   Skills    │  ← Normal card
└─────────────┘
```

### Hover State
```
┌─────────────┐
│   📚  ✨    │  ← Icon scales + sparkle
│   Skills    │  ← Card lifts + shadow
└─────────────┘
```

```
╔═════════════╗
║   📚   ✓    ║  ← Thicker border
║   Skills    ║  ← Checkmark indicator
╚═════════════╝
```
